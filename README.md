# Niryata Trading Private Limited

A modern, responsive website for Niryata Trading Private Limited - a professional trading and logistics company.

## About

Niryata Trading Private Limited is a professionally managed Trading and Logistics company, delivering reliable sourcing, trading, and end-to-end logistics solutions across international markets.

## Features

- **Modern Design**: Clean, professional design with teal/blue color scheme
- **Responsive Layout**: Mobile-friendly design that works on all devices
- **Multiple Pages**: 
  - Home page with hero section and services overview
  - About page with company information
  - Products page showcasing product categories
  - Contact page with form and company details
- **Built with Next.js 14**: Using the latest App Router
- **TypeScript**: Type-safe code throughout
- **Tailwind CSS**: Utility-first CSS framework for styling

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
